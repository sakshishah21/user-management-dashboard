# User Management Dashboard

A responsive User Management Dashboard built using React.js that allows users to **view, search, filter, sort, add, edit, and delete users** using the DummyJSON API.

---

# Features Implemented

## Users List Page

- Display users in table format
- Show Avatar, Full Name, Email, Phone, Company Name, Role
- Action buttons: View, Edit, Delete

---

## Search Functionality

- Search users by First Name
- Search users by Last Name
- Search users by Email

---

## Filters

- Filter users by Gender
- Filter users by Role

---

## Sorting

- Sort users by Name (A–Z / Z–A)
- Sort users by Age (Low–High / High–Low)

---

## Pagination

- 10 users per page implemented

---

## Add User

- Modal form for adding users
- Required field validation
- Clean UI form design

---

## Edit User

- Same modal used for edit
- Pre-filled form data

---

## Delete User

- Confirmation modal before deleting user

---

## User Details Page

- Full user profile view
- Sections:
  - Basic Information
  - Address Information
  - Company Information
  - Additional Information

---

## UI Features

- Fully responsive design (Mobile + Desktop)
- Custom Loader component
- Toast notifications
- Clean UI without any external UI library

---

## Tech Stack

- React.js (Functional Components)
- JavaScript (ES6+)
- React Router DOM
- Fetch API / Axios
- HTML5 + CSS3
- Custom CSS (No UI libraries used)

---

## API Used

DummyJSON Users API  
https://dummyjson.com/docs/users

# Setup & Run Instructions

## Create React App

- npx create-react-app user-management-dashboard
- npm start (run command)
